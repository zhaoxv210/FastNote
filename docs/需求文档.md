# 需求文档

## 1. 概述

FastNote（速记）是一个极简的本地 Markdown 笔记应用，基于 Tauri 2 + React 19 + Rust 构建。目标用户是需要快速记录、本地优先、无云依赖的知识工作者。

### 核心价值
- **零配置**：选择本地文件夹即用，无数据库、无账号
- **极速搜索**：全 vault 全文搜索，Ctrl+K 即搜
- **所见即所得**：Milkdown 编辑器，实时预览 Markdown

## 5. 数据流

```
用户操作 → React Event → Zustand Action → invoke() IPC
→ Rust Command → std::fs 操作 → Result → IPC 返回
→ Zustand State 更新 → React Re-render → UI 更新
```

### 典型流程：打开文件
1. 用户点击文件树中的文件名
2. FileTree 组件调用 `openFile(filePath)`
3. store 自动保存当前文件（如有脏内容）
4. `invoke("open_file", { vaultPath, filePath })` 发送到 Rust
5. Rust 读取文件内容并返回字符串
6. store 更新 `activeFile`, `activeContent`, `savedContent`
7. Editor 组件监听 `activeContent` 变化，更新 Milkdown 内容

### 典型流程：拖拽移动
1. `mousedown` 记录源路径，添加 `mousemove`/`mouseup` 监听器
2. 鼠标移动 > 6px → 开始拖拽：源半透明，创建浮动标签
3. `elementFromPoint` 检测目标元素，找到 `[data-tree-path]` 属性
4. 目标文件夹高亮（outline + 背景色）
5. `mouseup` → 调用 `moveItem(source, targetPath)`
6. `invoke("move_item", ...)` → Rust 执行 `fs::rename`
7. 返回新路径 → store 更新 `activeFile` 并重新加载树
